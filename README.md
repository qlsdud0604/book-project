# Book 프로젝트

## 기술 스택
* IDE
```
 - Visual Studio Code
 - IntelliJ IDEA
```
* Language
```
 - JavaScript
 - Java
```
* Front-end
```
 - React
 - Bootstrap
 - react-router-dom
```
* Back-end
```
 - Spring Boot
 - JPA
 - Maven
 - Lombok
```
* Database
```
 - H2
```
</br>

## 프로젝트 목적
* React를 사용하여 화면을 구성하고 컴포넌트 간의 동작 과정을 파악한다.

* Spring Boot로 서버를 구축하고 React와 통신하는 방법을 터득한다.
</br>


## 프로젝트 기능
* 글 작성 기능

* 글목록 조회 기능

* 글 상세 조회 기능

* 글 수정 기능

* 글 삭제 기능
</br>
